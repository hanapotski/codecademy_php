Echo
The echo function outputs strings. If you type

<?php
  echo "Hello!";
?>
PHP will output Hello!.

Make sure to end your line of PHP code with a semicolon.

<!DOCTYPE html>
<html>
    <head>
	</head>
	<body>
        <h1>
          <?php
            echo "I'm learning PHP"; 
          ?>
        </h1>
	</body>
</html>
